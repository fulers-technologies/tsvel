# @tsvel/logger

Standalone logging system with i18n support and decorators for the TSVEL Framework.

## Installation

```bash
npm install @tsvel/logger
# or
yarn add @tsvel/logger
# or
pnpm add @tsvel/logger
```

## Usage

```typescript
import { Logger } from '@tsvel/logger';

// Basic usage example
const example = new Logger();
```
