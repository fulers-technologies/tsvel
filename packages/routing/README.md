# @tsvel/routing

Framework-agnostic routing system with CLI and documentation support for the TSVEL Framework.

## Installation

```bash
npm install @tsvel/routing
# or
yarn add @tsvel/routing
# or
pnpm add @tsvel/routing
```

## Usage

```typescript
import { Router } from '@tsvel/routing';

// Basic usage example
const example = new Router();
```
