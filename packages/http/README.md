# @tsvel/http

HTTP utilities for server and client with Laravel-inspired Response class for the TSVEL Framework.

## Installation

```bash
npm install @tsvel/http
# or
yarn add @tsvel/http
# or
pnpm add @tsvel/http
```

## Usage

```typescript
import { Http } from '@tsvel/http';

// Basic usage example
const example = new Http();
```
