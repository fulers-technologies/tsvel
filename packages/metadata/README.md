# @tsvel/metadata

Standalone metadata management inspired by inversiland metadata for the TSVEL Framework.

## Installation

```bash
npm install @tsvel/metadata
# or
yarn add @tsvel/metadata
# or
pnpm add @tsvel/metadata
```

## Usage

```typescript
import { Metadata } from '@tsvel/metadata';

// Basic usage example
const example = new Metadata();
```
